Thank you for downloading the Evolved Mod! 
Follow these simple steps to install the mod and fix the internet crash.

[BEFORE YOU START]
It is highly recommended to make a backup copy of your original files 
before replacing them.


STEP 1: Install the Crash Fix & Loader
--------------------------------------
1. Copy the "version.dll" file from your download.
2. Paste it directly into your main game directory, right next to the "Feed and Grow.exe".
   Path: ...\Steam\steamapps\common\Feed and Grow Fish\

(This file fixes the engine crash on modern CPUs and keeps online features active!)


STEP 2: Install the Mod Features
--------------------------------
1. Copy the "Assembly-CSharp.dll" file from your download.
2. Go into the game data folder called "Feed and Grow_Data", then open the "Managed" folder.
3. Paste the file inside and overwrite the existing one.
   Path: ...\Feed and Grow Fish\Feed and Grow_Data\Managed\


STEP 3: Start the Game
----------------------
1. Make sure your internet connection is ON.
2. Launch the game normally through Steam.
3. Enjoy playing every single fish and boss on every map!
